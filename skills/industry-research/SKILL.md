# 行业研究 Skill

## 0. 核心原则

行业研究不能从“列公司”开始。

必须先回答：

```text
这个行业现在到底在交易什么产业变量？
```

然后判断：

```text
它现在处于概念期、验证期、订单期、放量期，还是成熟期？
```

最后再映射到：

```text
谁是真受益，谁是沾边，谁只是情绪扩散？
```

本 Skill 的目标不是写行业科普，而是把行业主题翻译成可验证、可交易、可反证的投资框架。

---

## 1. 触发条件

当用户出现以下表达时调用本 Skill：

- “研究一下 XXX 行业”
- “XXX 这条线怎么看”
- “玻璃基板/ADC/多肽/MLCC/存储/CPO/机器人/半导体材料有什么公司”
- “这个行业现在在交易什么”
- “这个板块的产业链怎么拆”
- “这个主题有没有类似牧原猪价/中芯ASP那种映射”
- “这个行业的核心验证指标是什么”
- “哪些公司是真受益，哪些是沾边”
- “这个行业贵不贵”
- “海外有什么可比”
- “A股怎么映射”

如果用户问单只股票，优先调用 `valuation-mapping`。  
如果用户给的是新闻或小作文，优先联动 `serenity-alpha`。  
如果用户问海外异动对 A 股影响，联动 `overseas-a-share-mapping`。  
如果用户问趋势健康和交易时点，联动 `gf-dma-health-index` 或 `horizontal-breakout`。

---

## 2. 必须先做当前数据校验

行业研究涉及当前产业进展、政策、价格、产能、订单、海外公司估值、A股市值和新闻时，必须获取最新数据。

不得凭记忆给出：

- 当前市场规模
- 当前价格
- 当前产能
- 当前供需
- 当前市值
- 当前估值
- 当前行业新闻
- 当前海外公司进度
- 当前公司订单和客户认证

需要校验的数据包括：

- 最新行业报告
- 公司公告和调研纪要
- 业绩预告和财报
- 海外龙头财报/电话会
- 产业链价格
- 库存、稼动率、出货量
- 资本开支
- 客户认证进度
- 政策和监管变化
- A股/港股/海外可比公司市值和估值

如果数据无法确认，必须明确说“当前数据不足”，并用情景假设替代，不能假装确定。

# 一、一句话结论

先用 3-5 句话说明：

1. 这个行业现在值不值得研究。
2. 当前市场交易的是需求爆发、供给瓶颈、价格上涨、国产替代、技术路线切换、资本开支，还是政策催化。
3. 当前处于概念期、验证期、订单期、放量期还是成熟期。
4. 最大预期差在哪里。
5. 最关键的验证指标是什么。

示例：

```text
玻璃基板不是“普通玻璃涨价”逻辑，而是 AI/HPC 推动先进封装大尺寸化后，市场在交易下一代封装载板/中介层路线的 0→1。当前处于验证期早段，不是利润兑现期。短期看送样、认证、试验线和设备订单，中期看 2028 年前后能否进入小批量/量产。
```

# 二、行业交易锚

必须先判断市场当前到底在交易什么。

常见行业交易锚：

| 交易锚 | 典型行业 |
|---|---|
| 需求爆发 | AI服务器、CPO、HBM、GLP-1、机器人 |
| 供给瓶颈 | 光模块、先进封装、半导体设备、电子布、MLCC |
| 价格上涨 | 猪价、锂价、硅片、存储、MLCC、电子化学品 |
| 国产替代 | 半导体设备、材料、EDA、信创、创新药出海 |
| 技术路线切换 | 玻璃基板、CPO、硅光、固态电池、Chiplet |
| 资本开支扩张 | 晶圆厂、存储扩产、数据中心、电网、燃机 |
| 政策催化 | AIC、低空经济、算力、医药创新、机器人 |
| 出海逻辑 | 创新药、CXO、工程机械、消费电子、汽车零部件 |
| 库存周期 | MLCC、存储、面板、猪周期、化工 |
| ROE修复 | 保险、银行、周期制造、消费龙头 |

输出格式：

```text
当前行业交易锚是：XXX。
不是在交易：XXX。
原因是：XXX。
```

必须区分：

- 表面叙事
- 真实利润来源
- 产业链卡点
- 资金实际交易方向
- 可验证产业指标

# 三、行业阶段判断

必须判断行业处在哪个阶段。

| 阶段 | 特征 | 市场交易方式 | 研究重点 |
|---|---|---|---|
| 概念期 | 巨头提路线、政策提出、技术想象空间大 | 炒主题、炒名字 | 谁有技术储备，谁只是沾边 |
| 验证期 | 送样、认证、中试线、试验线、客户测试 | 炒验证节点 | 谁进入客户体系，良率/成本/指标如何 |
| 订单期 | 小批量订单、产能建设、合同公告 | 炒收入弹性 | 订单金额、客户、ASP、交付周期 |
| 放量期 | 收入占比提升、利润兑现 | 炒业绩 | 毛利率、份额、净利率、现金流 |
| 成熟期 | 格局稳定、增速下降 | 炒估值和ROE | 分红、ROE、成本、份额 |
| 过热期 | 估值已经交易远期乐观情景 | 炒情绪 | 反证条件、兑现风险、交易结构 |

输出：

```text
当前行业阶段：概念期 / 验证期 / 订单期 / 放量期 / 成熟期 / 过热期。
判断依据：
1. ...
2. ...
3. ...
```

注意：

- 概念期不能按业绩股估值。
- 验证期重点看客户认证。
- 订单期重点看订单和收入确认。
- 放量期重点看毛利率和利润弹性。
- 过热期重点看反证条件和估值隐含预期。

|---|---|---|---|
| 下游应用 | | | |
| 客户资本开支 | | | |

重点回答：

# 五、技术路线和路线竞争

如果行业涉及技术变化，必须拆技术路线。

输出：

| 技术路线 | 优点 | 缺点 | 当前成熟度 | 代表公司 | 胜出概率 |
|---|---|---|---|---|---|

必须回答：

- 当前主流路线是什么？
- 新路线解决了什么问题？
- 新路线的成本、良率、可靠性是否可行？
- 旧路线是否还有扩展空间？
- 新路线是替代、补充，还是小众应用？
- 产业龙头是否真正采用？
- 商业化时间点在哪里？

示例：

玻璃基板不能简单说“替代 CoWoS”。更准确的研究方式是：

```text
玻璃基板是先进封装大尺寸化的潜在下一代载板/中介层路线，但短期更可能是补充路线而非全面替代。需要看 Intel、TSMC、Samsung、SK Absolics 等龙头的验证和量产节奏。
```

# 六、产业链拆解

必须按价值量和卡点拆，而不是只按上中下游粗略分类。

输出：

| 环节 | 核心产品/服务 | 价值量 | 壁垒 | 供需状态 | 代表公司 |
|---|---|---:|---|---|---|

进一步分层：

| 层级 | 定义 |
|---|---|
| 一阶受益 | 需求直接传导到收入和利润 |
| 二阶受益 | 通过设备、材料、零部件间接受益 |
| 三阶受益 | 概念相关，但收入传导弱 |
| 情绪扩散 | 只因名称、概念或历史题材被交易 |

必须回答：

- 哪个环节价值量最高？
- 哪个环节最卡脖子？
- 哪个环节最先兑现订单？
- 哪个环节利润率最高？
- 哪个环节竞争最激烈？
- 哪个环节最容易被市场误分类？

- 产能
- 交付周期

- 可靠性
- 资本开支

# 九、A股/港股映射

A股/港股公司必须分层，不得一锅端。

输出：

| 层级 | 公司 | 映射逻辑 | 真实性 | 弹性 | 风险 |
|---|---|---|---|---:|---|
| 核心 | | | | | |
| 次核心 | | | | | |
| 观察 | | | | | |
| 沾边 | | | | | |
| 纯情绪 | | | | | |

分层标准：

## 9.1 核心标的

满足至少两项：

- 有明确产品/客户/订单/认证/产能。
- 业务和行业核心瓶颈直接相关。
- 收入弹性能够改变财务报表。
- 公司公告或财报中已有实质进展。
- 不是单纯互动易概念。

## 9.2 次核心标的

满足：

- 有相关产品或技术储备。
- 产业链位置较近。
- 但收入占比、客户、订单尚不清晰。

## 9.3 沾边标的

特征：

- 公司名称或历史业务相关。
- 暂无明确收入或客户。
- 产业链传导距离较远。
- 主要适合情绪交易，不适合基本面核心池。

## 9.4 纯情绪标的

特征：

- 没有实质业务。
- 公司公告提示业务占比极低或无收入。
- 股价主要由题材和资金推动。

必须提醒：

```text
不要因为公司名称或行业标签相似，就认定为真受益。
```

# 十、估值映射和行业空间反推

行业研究也必须做估值映射。

不能只说 TAM 很大。必须反推：

```text
当前板块市值隐含了多大的未来收入、利润、销量、价格、订单、渗透率或ROE？
```

常用框架：

## 10.1 行业空间

```text
行业收入空间 = 下游需求量 × 单位价值量 × 渗透率
```

## 10.2 公司收入

```text
公司收入 = 行业收入空间 × 公司份额
```

## 10.3 公司利润

```text
公司净利润 = 公司收入 × 净利率
```

## 10.4 合理市值

```text
合理市值 = 公司净利润 × 合理PE
```

或：

```text
合理市值 = 公司收入 × 合理PS
```

## 10.5 当前市值反推

```text
当前市值隐含收入 = 当前市值 / 合理PS
当前市值隐含利润 = 当前市值 / 合理PE
当前市值隐含份额 = 隐含收入 / 行业收入空间
```

输出：

| 项目 | 保守 | 中性 | 乐观 |
|---|---:|---:|---:|
| 行业空间 | | | |
| 渗透率 | | | |
| 公司份额 | | | |
| 公司收入 | | | |
| 净利率 | | | |
| 净利润 | | | |
| 合理倍数 | | | |
| 合理市值 | | | |
| 当前市值是否已反映 | | | |

如果行业还在概念期或验证期，不能机械使用成熟期 PE，应使用“期权价值 + 里程碑验证”的框架。

# 十四、反证条件

必须写清楚什么情况说明行业逻辑错了。

常见反证：

- 下游需求不及预期。
- 价格没有上涨或持续下跌。
- 客户认证失败。
- 订单没有兑现。
- 行业产能过快扩张导致供给过剩。
- 良率/成本长期无法解决。
- 海外龙头路线改变。
- A股公司公告显示业务占比极低。
- 财报没有体现收入和利润。
- 毛利率下滑。
- 现金流恶化。
- 海外可比估值下修。
- 市场已经交易乐观情景但产业仍停留在概念期。
- 政策或监管导致估值折价扩大。

输出格式：

```text
如果出现以下情况，应下修行业判断或降级为观察：
1. ...
2. ...
3. ...
```

反证条件必须具体，不能只写“产业进展不及预期”。

当前行业属于：

# 十六、行业分类特殊处理

# 十七、输出风格要求

- 先结论，后推导。
- 用表格拆产业链和标的。
- 少写泛泛科普，多写交易锚和验证指标。
- 对不确定数据必须明确标注。
- 对当前市场数据必须引用最新来源。
- 不要因为题材热就默认看多。
- 不要因为长期空间大就忽略短期兑现。
- 必须区分“行业逻辑成立”和“当前股票可以买”。
- 必须区分“真受益”和“概念扩散”。
- 必须写反证条件。

---

# 十八、禁止项

- 禁止一上来列股票。
- 禁止只写行业空间，不写当前阶段。
- 禁止只说“受益于 AI/国产替代/景气复苏”。
- 禁止不拆产业链价值量和卡点。
- 禁止把 TAM 直接等同于公司收入。
- 禁止把概念期公司当作业绩兑现公司。
- 禁止把普通玻璃、普通材料、普通设备公司直接当核心受益。
- 禁止不做海外映射。
- 禁止不写验证指标。
- 禁止不写反证条件。
- 禁止只给“长期看好”而不给交易结论。
